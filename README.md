## üß† What I Learned
Building this without Bootstrap forced me to solve common layout problems manually:
1.  **The "Div Center" Challenge:** Mastered centering elements both horizontally and vertically using `display: flex` and `place-items: center`.
2.  **Z-Index Management:** managing stacking contexts in the Hero section to keep text readable above the background graphics.
3.  **Box Model:** Gaining total control over `padding`, `margin`, and `border-box` sizing to prevent layout shifts.
